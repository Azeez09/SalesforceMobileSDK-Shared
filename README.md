# Salesforce.com Mobile SDK Shared Repository
This repository contains artifacts that are shared between the different Salesforce Mobile SDK repositories.  For more information, please check out the [Salesforce Mobile SDK for Android](https://github.com/forcedotcom/SalesforceMobileSDK-Android/), or the [Salesforce Mobile SDK for iOS](https://github.com/forcedotcom/SalesforceMobileSDK-iOS).
